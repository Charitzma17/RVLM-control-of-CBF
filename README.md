# RVLM-control-of-CBF
Comprises of codes and sample data sets to run the codes. It has the following:
1. Fiber photometry analysis with simultaneous extraction of corresponding oxygen sensor data to analyse data during baseline, hypoxia and recovery
2. Vasodilation calculation using Full width half max calculation from the acquired two photon data
3. Smooth muscle cell calcium analysis from widefield experiments
4. Laser doppler flowmetry signal analysis with analysis of acquired electrocorticogram data
5. Analysis of the breathing data acquired from the Far infrared camera
6. Analysis of the ECG data acquired from the electric paw pads
7. AAV projection processing
8. HSV data analysis
9. Chemogenetic data analysis
