# RVLM-control-of-CBF
Hypoxia Fiber photometry data processing
1. Demultiplex fiber photometry signal
2. Calculate dF/F as Fsig-Fisosbestic/Fisosbestic
3. Extract time stamps for baseline, hypoxia, and recovery
4. Subtract mean baseline from the dF/F and LDF signal
5. Plot signals
6. Separate epochs and calculate cross correlation
7. Repeat for all mice and calculate averages for boxplots
